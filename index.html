Bienvenido

Esta pagina esta diseñada con el objetivo de presentar nuestro proyecto AlmaZen, una aplicación web para la gestión de inventarios de alimentos y productos diversos. A continuación, se describen los servicios que ofrecemos, información sobre la empresa y cómo contactarnos.

Servicios

Nuestro proyecto tiene como objetivo desarrollar una aplicación web intuitiva y eficiente que permita a usuarios individuales (personas naturales) y pequeños negocios gestionar y hacer seguimiento de su inventario de alimentos y productos diversos. La aplicación facilitará la organización y control del stock, proporcionando herramientas para el registro de entradas y salidas de productos, el seguimiento de fechas de caducidad, y la gestión de reposiciones y reabastecimientos. Además, contará con funcionalidades como el escaneo de códigos de barras para un registro automático y manual de productos, notificaciones para alertar sobre inventarios bajos o vencimientos próximos, y reportes detallados para optimizar la toma de decisiones en cuanto a compras y consumo.

Este sistema tiene como propósito mejorar la eficiencia operativa, reducir el riesgo de desabastecimientos y pérdidas, y permitir a los usuarios mantener un inventario bien organizado y actualizado, ya sea para uso personal o como parte de un negocio de pequeñas dimensiones.

Dentro de los servicios que ofrecemos se encuentran:
  • Registro de productos a traves de código de barra
  • Registro de productos de forma manual
  • Registro de productos por medio de un archivo CSV
  • Eliminación de productos
  • Actualización de productos
  • Organización de productos por categorías
  • Visualización de productos
  • Visualización de productos por categoría
  • Visualización de productos por fecha de vencimiento
  • Visualización de productos por cantidad
  • Sugerencias de productos a reabastecer
  • Notificaciones de productos próximos a vencer
  • Reportes de productos
